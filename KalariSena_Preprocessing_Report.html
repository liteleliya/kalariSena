KalariSena — Root-Height Preprocessing Summary

Pipeline applied to retargeted Unitree G1 (29-DOF) motion (Urumi_Sword_retarget_g1.csv, 1321 frames, 30 fps). Implements paper §3.2.1–3.3. Units: translations cm→m, angles deg→rad, g = 9.81 m/s².

z_min(q) = lowest body point vs. pelvis (forward kinematics). Grounded height = −z_min. Clearance = pelvis_height + z_min (0 = foot on floor, <0 = penetration).

Chosen parameter values

tau10.0015 m/framevelocity gate for drift; just above jitter floor (~1–2 mm/frame)

sg_window731 (odd)smoothing window; paper's 1/10≈131 over-smooths this long clip
sg_polyorder73cubic local fit

Values were set from a calibration of our data (velocity / clearance histograms + Otsu). Otsu's automatic splits (0.0065 / 0.137 / 0.0152) were too high because fast martial-arts motion has no clean jitter-vs-motion gap, so we kept the velocity gates low and only nudged the contact band up.

Steps

1 · Root Node Height Drift Correction (§3.2.1). Removes slow pelvis float/sink and rebuilds jumps. Velocity Ṗ(t)=P(t+1)−P(t); anchor first frame to grounded height; at contacts snap to −z_min; otherwise P̂(t)=P̂(t−1)+𝟙[Ṗ(t−1)>τ]·Ṗ(t−1) (add only genuine rises); jump descent filled with a gravity parabola; clamp penetration. Result: drift ~1.2 m → steady ~0.75 m; 3→0 penetrating frames.

2 · Minimum Body Height Constraint (§3.2.2, eq 3–5). z_min(q)=minᵢ zᵢ(q). Contact phases: p̂=p−z_min → lowest part exactly on floor (=0); airborne: keep but ≥0. Result: 1001/1321 contact frames flattened to exactly 0; min clearance 0.

3 · Temporal Propagation w/ Velocity Thresholding (§3.2.3, eq 6–7). De-jitter in time: P̂(t)=P̂(t−1)+Ṗ(t−1) only if |Ṗ|>τ₃, else hold; contacts stay anchored (noise-gate idea). Result: 149 jitter frames suppressed; temporally smooth.

4 · Jump Phase Detection (§3.2.4, eq 8). Local maxima (P'=0, P''<0, apex) / minima (P'=0, P''>0, landing). A segment is a real jump only if clearance > airborne_threshold (objective skip set for squats/stands). Result: 13 true jump segments found; mid-sequence bobs skipped.

5 · Parabolic Reconstruction of Jumps (§3.2.5, eq 9–12). Per jump keep apex y₀, landing y₁; free-fall y(t)=y₀−½g t²; airtime T=√(2(y₀−y₁)/g); sample Δtₖ=k/(N+1)·T; ascent = symmetric mirror. Result: 174 frames replaced by clean gravity arcs.

6 · Final Ground Penetration Correction (§3.2.6, eq 13). Safeguard: if z_min(q̂)<0 → p̂=p−z_min. Guarantees z_min≥0 everywhere. Result: 0 frames needed fixing (already clean).

7 · Savitzky–Golay Smoothing (§3.3). Per-channel SG filter on all 35 channels (root + 29 joints): local polynomial fit, removes jitter but preserves peaks (apexes/contacts). Window w odd, order p. Result: final smooth CSV; jump peaks and contacts preserved.

Pipeline

raw → S1 drift+parabola → S2 min-body-height → S3 temporal de-jitter
    → S4 jump detect → S5 parabolic jumps → S6 penetration clamp → S7 SG smoothing → final CSV

Tooling: yourdfpy (FK, Windows-friendly; Pinocchio has no Windows wheel), scipy (find_peaks, savgol_filter), meshcat (3D viewer). One-file runner: preprocessing/run_all.py (toggle stages with booleans).
